# Minesweeper Clone

*[WIP]*

A classic logic puzzle game implemented as a Minesweeper clone. This project was built entirely using the **Godot Engine**, providing a smooth and responsive gaming experience.

**Made by me** if you enjoy my game, please check out my [itch page](https://atomicmage.itch.io/)

## 🕹️ Features
*   **Classic Gameplay:** Enjoy the familiar challenge of revealing safe tiles and avoiding hidden bombs.
*   **Sandbox settings:** Set custom game parameters.
*   **Highscore:** Get to the top of your best scores.

## 🚀 Getting Started

This project is designed to run on Godot. Basic instructions or running the game can be found in the documentation folder.
